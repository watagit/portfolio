# Product Overview

個人ポートフォリオサイト - プロフェッショナルな経歴とプロジェクトを紹介

## Core Capabilities

- プロフィール表示 - 経歴、スキル、実績の紹介
- プロジェクト一覧 - 個人・業務プロジェクトのショーケース
- ソーシャルリンク - GitHub、SNSなどのプロフェッショナルプロファイル連携
- レスポンシブデザイン - モバイル/デスクトップ対応

## Target Use Cases

- 採用担当者・リクルーターへの経歴アピール
- 技術スキル・プロジェクト実績のデモンストレーション
- ソーシャルプレゼンスの統合

## Value Proposition

シンプルで洗練されたデザインで、技術者としてのアイデンティティとスキルセットを効果的に伝える
